Common Questions/Problems

  • Many examples of noise focus on visualizing the noise. Leading students to think that the noise() function is mostly when you want something to look like noise(). On the contrary, the noise function can be used any time you want good looking variation. Also, you probably want to shape the noise a bit so it doesn’t look like straight noise().

  • random() doesn’t take parameters but noise() does. Understanding what parameters to pass in is often confusing when you start using noise(). Once you get a hang of it, the parameters become the key to the power of noise().
